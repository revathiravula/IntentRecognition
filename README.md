# IntentRecognition
```sh
cd <directory where source and makefile are present>
download and install MicrosoftCognitiveServicesSpeech SDK from [here](https://docs.microsoft.com/en-gb/azure/cognitive-services/speech-service/quickstarts/setup-platform?tabs=macos%2Cubuntu%2Cdotnet%2Cjre%2Cmaven%2Cnodejs%2Cmac%2Cpypi&pivots=programming-language-cpp)
add SDK path to SPEECHSDK_ROOT in makefile
make
./IntentRecognition
```
